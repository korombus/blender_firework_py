# Environment
- Blender 2.9x

# Usage
1. Blenderを起動
1. Scriptingタブにて「run_external_script.py」を開く
1. 記述されている"FILE_ROOT_PATH"を自身の環境に合わせて変更
1. 「スクリプト実行ボタン」を押下
1. Animationタブにて「3Dビュー」を「レンダー」に変更
1. 再生ボタンを押して、アニメーションを確認
